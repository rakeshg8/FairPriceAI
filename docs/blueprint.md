# **App Name**: PriceWise AI

## Core Features:

- Product Data Input: Collect product details (name, photo, MRP) using input fields and image upload.
- AI Price Analysis: Analyze the product and break it down into components. Estimate the cost of each component using GenAI, considering market price, brand value, and material quality; and give an overall assessment. Format the AI tool response as JSON with cost breakdown, total estimated cost, verdict, and a plain-language explanation.
- Chat UI Display: Present the analysis in a chat-style interface, including assistant messages, a pie chart visualizing the cost breakdown, and an image viewer for the product photo.
- Image Storage: Handle image uploads to Firebase Storage.
- Loading and Error Handling: Handle loading states and errors gracefully.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to evoke trust and sophistication, while still keeping the vibe modern and innovative.
- Background color: Light purple (#F2EEFA), very lightly tinted and desaturated to keep the focus on the content.
- Accent color: A vibrant magenta (#E000A4) to draw attention to key actions and information in the app.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clean, minimalist icons to represent different product components and analysis results.
- Implement a responsive layout to ensure optimal viewing on various devices.
- Subtle transitions and animations to enhance the user experience during loading and data updates.
